This is a python-based simulation that demonstrates temporal inversion by recording 
 video footage and playing it backwards, allowing users to experience time-reversed
 interactions akin to those seen in Christopher Nolan's 2020 film "Tenet".

## Features 

- **Real-time Recording**: Capture video with live recording indicator and status display.
- **Temporal Inversion**: Plays recording in reverse, creating the Tenety-style time reversal effect.
- **Custom Components**: Use custom components in your MDX files.
- **Responsive Design**: The website is responsive and works on all devices.

## Technologies 

- **Python**: The programming language driving this project's temporal magic.
- **OpenCV**: Handles capturing video from the camera and processing those video streams.
- **NumPy**: Provides support for handling mathematical operations, specifically working with arrays of video data.

## Getting Started

To get started with this project, you can clone the repository and install the
dependencies:

```bash
git clone
cd next-mdx-portfolio
npm install
```

Once the dependencies are installed, you can run the development server:

```bash
npm run dev
```

The development server will start at `http://localhost:3000`.
